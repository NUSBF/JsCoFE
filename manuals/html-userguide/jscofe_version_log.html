    11. CCP4 Cloud Version Log¶

    Version 1.7.005

    Released on –.–.2022 with CCP4 8.0.003

    Changes and new features

    1. Project can be now organised in custom lists
    2. shared and imported projects maintain their original folder paths, starting from owner’s name
    3. Fixed bug causing crashes when job was rerun with last used parameters from a lower-version job
    4. Added Structure Prediction task based on AlphaFold-2
    5. Fixed bug in SHELX Auto-EP via Crank-2 associated with making PRASA the default choice for structure determination
    6. MorDA task can now use custom search models

    Version 1.7.004

    Released on 10.06.2022 with CCP4 8.0.002

    Changes and new features

    1. Project can be now organised in Folders
    2. Added new tasks Slice for splitiing models in domains, and Slice-n-dice for splitting followed by auto-MR with Phaser
    3. Covalent/metal link identification switch added in Refmac task
    4. Users can be now retired by admin with moving their projects to successor
    5. Fixed a bug that could cause a number cruncher halt if tasks MrBump, Ensemble preparation from sequences, Ensemple preparation from coordinates, Interactive model preparation with CCP4mg are forcefully terminated from the Project Page

    Version 1.7.003

    Released on 01.05.2022 with CCP4 8.0.001

    Changes and new features

    1. Fixed a bug in Import files task blocking import of XDS files
    2. Fixed a bug in handling AFDB models, causing MR failure in certain cases

    Version 1.7.002

    Released on 27.04.2022 with CCP4 8.0.000

    Changes and new features

    1. New task Prepare Complex MR Model added
    2. New task Model building with Modelcraft added
    3. New style built-in tutorials
    4. Projects can be cloned now

    Version 1.7.001

    Released on 29.11.2021 with CCP4 7.1.018

    Changes and new features

    1. Fixed bugs causing problems with https connections

    Version 1.6.024

    Released on 11.11.2021 with CCP4 7.1.017

    Changes and new features

    1. Fixed a bug which could cause project loss at insertion of remarks in the middle of a tree branch
    2. Improved navigation in Help window
    3. Documentation updates on Buccaneer, CCP4Build, Auspex and a number of other tasks
    4. External documentation links are now opened in new browser’s tabs
    5. Updates and fixes in MrBump, MrParse and Ensemble preparation with CCP4mg related to working with the AlphaFold database
    6. Fixed bug causing Job Dialog malfunction if protein and rna data types are mixed in the project
    7. Fixed bug causeing failure at importing merged .mtz files with phase information but without mean amplitudes.
    8. B-factors are automatically corrected in files from AFDB and Rosetta-Fold
    9. Demo projects may be linked directly from HTML documents
    10. Fixed bug which could cause project loss at inserting remarks while project contains tasks in exiting state
    11. Fixed bug which could cause project loss at archiving jobs at certain project tree configurations
    12. Remarks can be converted into web links (new option in the remark’s icon menu)
    13. Remarks’ icon menu is now scrollable and adjusting to the job dialog height
    14. Project ID can now be changed at renaming

    Version 1.6.023

    Released on 13.09.2021 with CCP4 7.1.016

    Changes and new features

    1. baverage and edsats analysis added to quality report section of various tasks
    2. Added disagnostics on non-existing PDB codes in PDB Import task
    3. Dimple task can now work with mean amplitudes
    4. Extended functionality, resource management and dimple task in cloudrun utility
    5. Initiation of Autostart projects simplified to the choice between few templates
    6. simbad task now searches all compatible space space groups (new option)
    7. fitligand - numbering of multiple ligands fixed
    8. refmac5 interface and Verdict update
    9. Table 1 added to quality report
    10. Fixed bug causing server crash at stopping tasks running on disconnected back-ends
    11. Improved stability in shared projects, which was showing up at high latency of network communication
    12. Fixed a bug in MrBump task causing the task to crash if no solution is found
    13. All Workflows updated to auto-generate the 3-letter ligand code if not provided or if on excluded list
    14. New Simple MR Workflow added.
    15. Fixed a bug causing malfunction when using Safari 14
    16. Documentation updates (Fragment-Based Molecular Replacement)
    17. Fixed a bug preventing deletion of old projects
    18. Fixed a bug in parrot and shelx-mr tasks preventing density modifciation immediately after hop-on import
    19. Fixed a bug due to which open dialogs could persist when switching between principal pages (Project, Account etc.)
    20. AlphaFold-2 support introduced in MrBump and Ensemble from Sequence tasks
    21. Coot task, if started from Job Dialog and produced no changes, is now automatically deleted.
    22. Hydrogens with zero occupancy may be now removed in the deposition task
    23. Project sharing/unsharing is now confirmed and notified by e-mail
    24. MrParse task added
    25. Sequence-lee and homology-based loop building options added to the Arp/wArp task
    26. Framework for exposing optional, 3rd-party licensed components to end users is added to CCP4 Cloud

    Version 1.6.022

    Released on 28.05.2021 with CCP4 7.1.014

    Changes and new features

    1. cloudrun utility for making projects, uploading data and starting tasks from command prompt
    2. Fixed bug in stats bookkeeping
    3. Added automatic workflow for refinement and ligand fitting
    4. Fixed bug causing corruption of shared projects in certain circumstances
    5. Introduced hot tips in front page
    6. Fixed bug causing task running spinner to keep going after task was terminated for certain type of tasks
    7. Multiple target sequences are allowed in MrBump auto-MR pipeline, as well as auto-MR and auto-EP workflows
    8. Documentation updates
    9. Fixed bug which did not allow to unjoin shared projects
    10. MrBump task modified
    11. Fixed bug in making PDB deposition package from Buster data
    12. Deposition task will now ask for target sequence if it was not given in the Project.
    13. Sort order of List of Projects now persists.
    14. It is now possible to specify image range and block size at working with HDF5 files in Xia-2 task
    15. Refinement with Buster no longer requires a preliminary Refmac run for producing a suitable PDB file

    Version 1.6.021

    Released on 15.04.2021 with CCP4 7.1.013

    Changes and new features

    1. Deleting running jobs made made a safe regular operation
    2. Improved stability of the PDB deposition task
    3. Fixed bug in importing expected sequences from the PDB (sequences in imported PDB headers)
    4. CCP4go task is retired
    5. Autosolve project mode converted to Autostart mode, and Expert mode converted to Standard mode
    6. Introduced Automatic Workflows
    7. Documentation updates

    Version 1.6.020

    Released on 15.03.2021 with CCP4 7.1.012

    Changes and new features

    1. Added Arcimboldo task for Fragment-based Molecuar Replacement
    2. Multiple GUI bugs fixed
    3. Fixed a bug breaking import of certain project files on MS Windows systems
    4. Buster 2.10.4 adopted
    5. Fixed a bug causing CCP4 Cloud to stall after running MR ensemble preparation tasks under certain circumstances
    6. Ligand libraries with duplicate component lists (old bugs elsewhere) can now be imported
    7. Crank-2 has new parameter “parallel model building depth” to balance between model building efficiency and speed
    8. Fixed a bug in Coot task on Windows causing the loss of backup copies
    9. Deposition task can now work with Buster output directly
    10. Fixed a bug due to which CCP4 Cloud erroneously asked for CPU quota in desktop mode
    11. Fixed bug in parsing newer HHPRED files (affected the Model preparation from alignment task)

    Version 1.6.019

    Released on 19.02.2021 with CCP4 7.1.011

    Changes and new features

    1. Added Import & Replace task, combining generic import and revision editing
    2. Specification of wavelength made compulsory on .sca file import
    3. Minor changes in task names and re-arangements in Task List
    4. Coot task now loads revision’s heavy-atom substructure automatically
    5. Anomalous maps are calculated automatically in Refmac and Buster tasks
    6. Projects may be shared from Project List, without opening the Project
    7. Hop on and Import & Replace tasks can import from Cloud storage
    8. Fixed bug in Gemmi task blocking writing results in output file at certain circumstances
    9. Fixed bug in Pisa task blocking visualisation of NCS-mates in UglyMol
    10. Jobs cannot be moved up the Job Tree now
    11. Fixed bug blocking auto-launch of Refmac after Coot finishes via Exit and refine in shared projects
    12. Fixed bugs in Task Dock which were causing instability
    13. Fixed bug in import of phases, causing misinterpretation of direct and difference maps
    14. Project summaries and scores added in the List of Projects
    15. Gelly options added to Buster task

    Released on 15.01.2021 with CCP4 7.1.010

    Changes and new features

    1. Verdict section added to Simbad task
    2. Fixed a bug causing loss of running jobs at server restart
    3. Added a toolbar button for switching between single and multiple selection in project tree
    4. Fixed a bug in Phaser-MR task, causing incorrect specification of Space Group in report pages
    5. Fixed a bug in Make Ligand task, causing failure if structure revision is produced in the task-contained branch
    6. Fixed a bug in Prepare MR Model from XYZ task, causing incorrect identification of sequence identity for nucleic acid models
    7. Added import Scalepack files (.sca)
    8. Fixed a bug in preparing DNA/RNA MR models, affecting sequence identity calculations
    9. Guided data import made optional, may be switched on/off in the Preferences section of the My Account page
    10. Fixed bug in unjoining shared projects
    11. Fixed bug causing duplication of input data supplied to Coot in certain circumstances
    12. Introduced configurable task dock for favourite tasks
    13. Add project function complemented with the choice of project type: autosolve, expert, or migration from elsewhere

    Version 1.6.017

    Released on 08.12.2020 with CCP4 7.1.009

    Changes and new features

    1. Fixed a bug blocking addition of new tasks after placing Remark at project’s top
    2. Change Resolution task fixed to allow the subsequent job tree development
    3. Fixed a bug in Phaser-MR task, causing the task to fail under certain circumstances
    4. Import from PDB and Model Preparation from Alignment Data tasks will now switch to downloading CIF-formatted files from the PDB if PDB format for the requested PDB entry is not available (typical for large structures with 2-character chain IDs)
    5. CCP4mg buttons in task reports are now hidden when no CCP4 Cloud Client is running
    6. CCP4Build task can be stopped gracefully, without losing already calculated results
    7. Excessively long branches may be archived as folders for convenience
    8. Documentation updates

    Changes and new features

    1. VDW repulsion weight parameter added in Refmac task
    2. Refmac verdict section updated
    3. Map selection added in FitLigand and FitWaters tasks
    4. New user preferences: login into last project, default window sizes for job dialogs and viewers
    5. Stalled jobs can be now pushed with button Refresh in the project page toolbar
    6. Coot task uses the last backup copy if user forgets to save coordinates
    7. New menu item: Load from Cloud backup, introduced in Coot, which allows a user to restore from backup copy if previous Coot Cloud task finished unsuccessfully
    8. Fixed bug in Buster, because of which the task did not pass ligand information on descendants
    9. Small bug fixes in Edit Revision
    10. Automatic clean-up of Front-End’s Job Register added

    Released on 14.10.2020 with CCP4 7.1.005

    Changes and new features

    1. Gemmi task added for custom manipulation with atomic coordinates
    2. Refmac verdict section updated
    3. ED fit analysis improvements in CCP4Build, leading to more stable convergence
    4. Documentation updates (DM with Acorn)
    5. Crank-2 report updated and documented

    Version 1.6.014

    Released on 14.09.2020 with CCP4 7.1.004

    Changes and new features

    1. Lorestr can now work with partially fragmented models (e.g., after automatic model building)
    2. Verdict sections aded to Lorestr and Dimple reports
    3. Fixed a bug in MrBump, which was causing report page corruption
    4. Fixed a bug in CCP4Build causing stop at mtz files with rare format features
    5. Resolved the cross-origin conflict with local client, causing corruption in report pages of tasks running on user devices, such as Arp/wArp
    6. Fixed bugs in Buster task, which caused task failures under specific conditions
    7. Option for removing coordinates added to edit revision tasks
    8. Video tutorials added in Documentation
    9. Port numbers for CCP4 Cloud Client made stable, which allows password management in users browsers
    10. Documentation updated ( Density modification, DM with Parrot)

    Version 1.6.013

    Released on 11.08.2020 with CCP4 7.1.003

    Changes and new features

    1. Shared Projects are now joined, rather than imported. This means that all users, who share a project, are working on the same copy of it. Simultaneous work of several users on shared projects is supported. Only project owners can share, delete or rename their project, other users can join or unjoin them in their Project Lists.
    2. Documentation updated (DIMPLE, Automatic Image Processing with Xia-2, Structure Superposition with LsqKab)
    3. DIMPLE task optimised by eliminating the trailing zero-cycle run of Refmac.
    4. Import of sequences from the PDB now produces sequences from both atomic coordinates (may be used for automatic construction of the corresponding structure revision) and from header annotation (more suitable for making MR search models).
    5. XYZ Utils can now import sequences from atomic coordinates, task interface changed
    6. Lorestr can now work on structures with ligand molecules

    Version 1.6.012

    Released on 03.07.2020 with CCP4 7.1.002

    Changes and new features

    1. CROSSEC task (anomalous scattering factors calculations) now operates with both wavelengths and photon energies. In addition to the general plot, anomalous scattering factors may be calculated also for up to 6 selected wavelengths or photon energies
    2. User guidance sections (Verdicts) are added to ASU, Phaser, CCP4Build and Refmac task reports
    3. Documentation updated
    4. Fixed a bug in the import procedure for HHPRED alignments, related to mispositioning of residue numbers under certain circumstances
    5. CCP4BUILD modified with the purpose to improve convergence and reproduceability of results at repeat runs.
    6. Fixed a bug in the import procedure for merged MTZ files, related to python2 / python3 code compatibility
    7. Added new task for importing sequences by copy-pasting (Task List Section Data Import)
    8. Cloud path, used in tasks importing data from Cloud storage, is passed as current path to task clones and descendants
    9. Fixed a bug, preventing import of certain types of ligand description files and ligand dictionaries
