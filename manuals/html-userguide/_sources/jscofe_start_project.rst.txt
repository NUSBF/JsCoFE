
====================
Work with CCP4 Cloud
====================

This article describes how to start a project and load data in CCP4 Cloud.

To begin, ensure that you have a `CCP4 Cloud account <https://cloud.ccp4.ac.uk/>`_ 
or have installed the **CCP4 Software Suite (version 7.1 and above)** 
(`Download here <https://www.ccp4.ac.uk/download/>`_).

------------------
Starting a Project
------------------

Step-by-step instructions on how to start a project in CCP4 Cloud can be found `here <https://scribehow.com/shared/Creating_project_and_importing_data_in_CCP4_Cloud__v2Gymj96RbarVwSOwL7C6Q>`_.

In CCP4 Cloud, work is organized into `Projects <https://cloud.ccp4.ac.uk/manuals/html-userguide/jscofe_myprojects.html>`_.

.. |addjob| image:: images/add.png
   :width: 16
   :height: 16

Press the |addjob| button above the project list to create your first project. Specify a short project name and a more descriptive title when prompted.

Manual mode
+++++++++++

Standard mode provides full manual access to operating all tasks in the 
CCP4 Cloud `task list <https://cloud.ccp4.ac.uk/manuals/html-userguide/jscofe_tasklist.html>`_.
The system learns your preferences in the course of your work and will
suggest new tasks in accordance with your habits. Automatically suggested tasks
appear in the Suggested tab of the *Task list*.

CCP4 Cloud includes several `data import <https://cloud.ccp4.ac.uk/manuals/html-taskref/Data_Import.html>`__ tasks:


1. `Data Import <https://cloud.ccp4.ac.uk/manuals/html-taskref/doc.task.Import.html>`_
2. `Import Sequence(s) by Copy-Paste <https://cloud.ccp4.ac.uk/manuals/html-taskref/doc.task.ImportSeqCP.html>`_
3. `Import from PDB/AFDB <https://cloud.ccp4.ac.uk/manuals/html-taskref/doc.task.ImportPDB.html>`_
4. `Import from Cloud Storage <https://cloud.ccp4.ac.uk/manuals/html-taskref/doc.task.CloudImport.html>`_
5. `Import & Replace <https://cloud.ccp4.ac.uk/manuals/html-taskref/doc.task.ImportReplace.html>`_
6. `Hop on CCP4 Cloud <https://cloud.ccp4.ac.uk/manuals/html-taskref/doc.task.Migrate.html>`_
7. `Fetch Diffraction Images from WWW repositories <https://cloud.ccp4.ac.uk/manuals/html-taskref/doc.task.FetchData.html>`_
8. `Push to Cloud Storage <https://cloud.ccp4.ac.uk/manuals/html-taskref/doc.task.PushToCloud.html>`_


Plans 1-4: Automated Structure Solution 
++++++++++++++++++++++++++++++++++++++++

Project plans correlate to existing `automatic workflows <https://cloud.ccp4.ac.uk/manuals/html-taskref/Workflows.html>`_

Available project plans include:

1. **Molecular Replacement using AlphaFold model**
2. **Molecular Replacement using structure databases**
3. **Molecular Replacement using a known model**
4. **Automatic Experimental Phasing**

After selecting a plan, you will be prompted to upload data. If you have no data but wish to practice, choose **CCP4 Cloud storage** under **Import data from**.

`Automation in CCP4 Cloud: Workflows <https://www.youtube.com/v/ph2rxQSYiE0>`_

Plan 5. Import already solved structure for completion
+++++++++++++++++++++++++++++++++++++++++++++++++++++++

This plan allows for a quick start from a phased (possibly partially built and refined) 
structure or heavy-atom substructure for further refinement and model building.
This mode features a task list optimized to essential tasks for structure completion
(can be switched to the full list). You can switch to the full task list at any time.

You will be directed to the `Hop on task <https://cloud.ccp4.ac.uk/manuals/html-taskref/doc.task.Migrate.html>`_, 
where you should provide reflection data, phases, atomic coordinates, and ligand descriptions 
(if available).
