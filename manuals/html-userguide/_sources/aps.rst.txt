
========================================================================
Transferring Projects Between Remote and Local CCP4 Cloud Installations
========================================================================

**Export projects from the web CCP4 Cloud** by using the |Export| button on the "My Projects" page.

1. Select the project you wish to export by left-clicking on it.

.. image:: images/aps_1.png

2. Click the |Export| button.

3. Click the download button.

.. image:: images/aps_2.png
   :scale: 40%

4. Choose a folder to save the CCP4 Cloud project.

5. After downloading the project to your local machine, locate the cloud icon in the Activities or Software menu (depending on your Linux version). Click on the |desktop| CCP4 Cloud desktop icon or type ``ccp4cloud-desktop.sh`` in the terminal.

6. Upload the project to the |local| local CCP4 Cloud installation using the |Import| button above the project list.

.. image:: images/aps_3.png
   :scale: 40%

   |local| in the top-right corner indicates that you are using a local installation of CCP4 Cloud.

7. Once imported, you can continue working on your project locally.

.. image:: images/aps_4.png

++++++++++++++++++++++
Important Notes
++++++++++++++++++++++

.. note:: You can export projects from a local installation using the |Export| button and later re-import them to the CCP4 Cloud server once it is available.

.. warning:: Log in to the |ccp4cloud_remote| CCP4 Cloud Remote and delete (or rename) all previously exported projects.

To log in, go to https://cloud.ccp4.ac.uk/ or locate the cloud icon in the Activities or Software menu. Click on the |ccp4cloud_remote| CCP4 Cloud Remote icon or type ``ccp4cloud-remote.sh`` in the terminal.

**To delete a project**:
- Select the project (left-click) in the project list.
- Click the |delete| delete button. 
  Be cautious: deleted projects cannot be restored.

**To rename a project**:
- Select the project (left-click) in the project list.
- Click the |rename| rename button.
  Ensure you change the Project ID, as duplicate IDs are not allowed in your CCP4 Cloud account.

.. image:: images/aps_6.png
   :scale: 50%

.. image:: images/aps_5.png
   :scale: 50%

Once renamed or deleted, you can re-import projects back to the CCP4 Cloud server:

1. Click the |Export| button.

2. Click the download button.

3. Choose a folder to save the CCP4 Cloud project.

4. Go to https://cloud.ccp4.ac.uk/ or locate the cloud icon in the Activities or Software menu. Click on the |ccp4cloud_remote| CCP4 Cloud Remote icon or type ``ccp4cloud-remote.sh`` in the terminal.

5. Upload the project to your CCP4 Cloud server account using the |Import| button.

6. Once imported, you can continue working on your project on the server.
