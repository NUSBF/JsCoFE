    10. CCP4 Cloud Education Capabilities¶

    CCP4 Cloud is not only a powerful tool for managing crystallographic projects but also an excellent platform for learning about the CCP4 tasks you are running.

    10.1. CCP4 Cloud Tutorials¶

    On the My Projects page toolbox, you will find the Tutorial icon. Clicking on it allows you to import tutorials or demo projects of interest to your project page.

    Tutorials include examples designed to demonstrate structure solution techniques in CCP4 Cloud. Tutorials are presented as seed projects with pre-loaded data, which can be imported into your account. Each tutorial includes step-by-step instructions to guide you through the required project development stages.

    You can find a list of all available tutorials here.

    Demo projects represent already developed job trees.

    10.2. Task Reference¶

    All tasks runnable in CCP4 Cloud include reference documentation. You can access task documentation by clicking the Task Documentation button in the Job dialog window.

    The main purpose of task documentation is to provide knowledge about CCP4 task capabilities and to enhance your experience with CCP4 tasks. Task documentation not only guides you in utilizing tasks more effectively but also expands your knowledge base about methods used to complete tasks.

    Tip

    You can access additional CCP4 Cloud task documentation by clicking Contents in the upper-right corner of the window. The contents page includes a list of all available task documentation.

    10.2.1. Questions and Suggestions¶

    At the end of both the User Guide and Task Documentation contents pages, you will find the CCP4 Cloud Reception. By clicking it, you will be redirected to a mail form where you can submit questions or suggestions about working in CCP4 Cloud, as well as feedback on task documentation content.
