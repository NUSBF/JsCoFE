BUSTER structure refinement package was provided by Global Phasing Limited (GPhL)

For more information, please visit BUSTER Documentation and BUSTER wiki pages.

  • R-factor definitions in BUSTER
  • Geometry restraints in BUSTER
  • Direct use of weighted Force Field or Quantum Chemical Energy for ligands in BUSTER
  • Rhofit - automated ligand fitter
  • buster-report - analytical report generation
  • Pipedream - automated pipeline package
  • mmCIF generation - for deposition
