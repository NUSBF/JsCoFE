2.2. Import Sequence(s) by Copy-Paste¶

This task allows a user to import sequence by copy-pasting them from, e.g., other web pages. All instructions are given in the task interface.

Note

several sequences may be copy-pasted in one go, if all of them belong to the same type (protein, DNA or RNA)

Note

CCP4 Cloud will assign sequence names using template name given on top of the interface. Default name is sequence, threfore, sequence names will be chosen sequence_01, sequence_02, … for 1st, 2nd … sequence given. This may be inconvenient for further identification, therefore, consider chosing meaningful template names and not importing more than one sequence a time.
