2.3. Import from PDB¶

This task allows importing coordinats, sequence and reflection files from Protein Data Bank. Improt from PDB task can create Structure Revision if the Structure revision box was checked.

Warning

Check the Structure revision box only if you are going to use the PDB data as Structure Revision for the following tasks

Note

The tasks in the Cloud use the last created Structure revision in the branch of the project tree. If structure revision was created in a middle of a project tree it replaces the previous Structure revision. The newly created Structure revision will be used as input for the following task. Please, make sure that the relevant data was selected in the Structure revision section for the following task.
