6.1.1. Find and prepare MR models with MrParse¶

MrParse takes a protein sequence file and searches for homologs using PHMMER for searching sequence databases for sequence homologs, and for making sequence alignments. It implements methods using probabilistic models called profile hidden Markov model, which is often considered to be the most sensitive sequence-comparison method.
